- ✅ **VISA, 마스터** → 호텔, 식당, 백화점 다 됨

- 팁 (식당 5~10%)
- 길거리 음식, 기념품 노점

- **트래블월렛 / 트래블로그** 카드 → 환율 우대 + ATM 무료
